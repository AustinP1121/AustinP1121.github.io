◆ peripheral_id_type

enum peripheral_id_type

Enum containing bit locations for each peripheral within the clock enable registers.

Each peripheral clock is enabled or disabled using the following registers:

  • RCC_APBENR
  • RCC_APB1ENR
  • RCC_APB2ENR

Ex. the clock enable bit for Port A is located at location 2 within the RCC_APB2ENR register.
